## Installation

- `git clone https://github.com/jenry23/Exam.git`
- `cd laravel-vue-crud-starter/`
- `composer install`
- `cp .env.example .env`
- Update `.env` and set your database credentials
- `php artisan key:generate`
- `php artisan migrate`
- `php artisan db:seed`
- `npm install`
- `npm run dev`
- `php artisan serve`
